# 🏸 배드민턴 운영용 사이트

> **코트 배치 + 참석자 관리 + 참여 횟수 누적**을 한 화면에서 처리하는 정적 웹앱입니다.

## 주요 기능

- ID 기반 참석자 관리 및 동명이인 구분
- 게임 횟수와 직전 경기 여부를 고려한 자동 배치
- 전체 참석자 목록에서 코트 슬롯으로 드래그 앤 드롭
  - PC: 마우스 드래그
  - 태블릿·모바일: 터치 드래그
- 현재 코트에 배치된 참가자를 별도 영역으로 표시
- 최근 상태 실행 취소
- 브라우저 `localStorage` 자동 저장

## 사용 방법

1. 참석자 이름을 쉼표 또는 줄바꿈으로 입력합니다.
2. 전체 참석자 목록의 이름을 원하는 코트 자리로 끌어 놓습니다.
3. 모바일에서는 이름을 누른 채 움직여 코트 자리에 놓습니다.
4. 드래그가 어려운 환경에서는 코트 자리를 먼저 선택한 뒤 참석자 이름을 누를 수도 있습니다.
5. 게임 종료 시 참여 횟수가 반영되고 해당 코트가 비워집니다.

## 데이터 저장

모든 데이터는 현재 브라우저의 `localStorage`에 저장됩니다. 브라우저 데이터 삭제 시 저장 내용이 사라질 수 있습니다.
