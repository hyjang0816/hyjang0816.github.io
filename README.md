# 🏸 배드민턴 운영용 사이트

> 코트 배치, 참석자 관리, 참여 횟수 누적을 한 화면에서 처리하는 정적 웹앱입니다.

## 주요 기능

- ID 기반 참석자 관리 및 동명이인 구분
- 참석자별 A~D 실력 등급 관리
- 게임 횟수와 실력 등급을 고려한 자동 배치
- 전체 참석자 목록에서 코트 슬롯으로 드래그 앤 드롭
  - PC: 마우스로 이름 영역을 끌어 놓기
  - 태블릿·모바일: 이름 영역을 누른 채 이동
  - 화면 위·아래 가장자리 접근 시 자동 스크롤
- 현재 코트에 배치된 참가자를 별도 영역으로 표시
- 코트별 또는 전체 게임 종료 시 참여 횟수 자동 반영
- 최근 10개 상태 실행 취소
- 브라우저 `localStorage` 자동 저장

## 사용 방법

1. 참석자 이름을 쉼표 또는 줄바꿈으로 입력합니다.
2. 전체 참석자 목록의 이름을 원하는 코트 자리로 끌어 놓습니다.
3. 모바일에서는 이름 영역을 누른 채 움직여 코트 자리에 놓습니다.
4. 먼 위치로 이동할 때는 화면 위·아래 가장자리까지 끌어 자동 스크롤합니다.
5. 드래그가 어려운 환경에서는 코트 자리를 먼저 선택한 뒤 참석자 이름을 누릅니다.
6. 게임 종료 시 참여 횟수가 반영되고 해당 코트가 비워집니다.

## 실행

별도 빌드 없이 `index.html`을 열거나 아래 명령으로 실행할 수 있습니다.

```bash
python3 -m http.server 8000
```

## 데이터 저장

데이터는 `badminton-ops-site-html-v2` 키로 현재 브라우저의 `localStorage`에 저장됩니다. 기존 v1 데이터는 최초 실행 시 자동 변환됩니다. 브라우저 데이터 삭제 시 저장 내용이 사라질 수 있습니다.
